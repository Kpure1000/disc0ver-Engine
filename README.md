# disc0ver-Engine
## 项目简介

近期在看 LearnOpenGL，准备边看边写一个工程，最后整合为一个渲染引擎。

任务列表：

- [x] 基础图元绘制
- [x] 基础图元纹理加载
- [x] 模型加载
- [ ] 光照 shader 实现
- [ ] GUI实现
- [ ] 光线追踪
